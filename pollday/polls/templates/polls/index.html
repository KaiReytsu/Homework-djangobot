Ответы на вопросы

  1. Как создать проект?

    2. Для создания проекта в папке с проектом надо развернуть виртуальную среду. Есть разные способы, я использую pipenv:
    Создаю папку Pipfile в папке проекта
    Запускаю pipenv shell
    если успешно, в консоле перед адресом папки будет указано: (название проекта)/адрес проекта/
    утсновить django: pipenv install Django

  2. Как создать приложение?

    3. django-admin startproject наименование приложения

  3. Как запустить тестовый веб-сервер?

    4. Запуск производится из папки с файлом manage.py. Для переход использовать команду cd название папки Запуск на linux: python3 manage.py runserver (5000 пример запуска по заданному порту)

  4. Как открыть на своем компьютере свой сайт на Джанго?

    5. В адресной строке можно ввести локальный ip адрес 127.0.0.1:порт
    Так же при запуске тестового веб-сервера можно пройти по ссылке, которая будет указана в консоле

  5. Как узнать, что мы допустили ошибку при редактировании файлов проекта?

    6. Ошибка будет выведена в консоле, так же, возможно, тестовый сайт будет отображать ошибку 404

  6. Как вывести на страницу HTML-код?

    7. HTML страница может быть расположена в папке "templates", чтобы программа могла найти шаблон, для этого надо прописать в программе какой шаблон ей надо использовать, так же можно разместить код в файле views.py, HttpResponse('html код')

  7. Почему команда запуска веб-сервера "ничего не пишет"?

    8. Она пишет:
    Watching for file changes with StatReloader
    Performing system checks...

    System check identified no issues (0 silenced).
    March 16, 2022 - 10:09:22
    Django version 4.0.3, using settings 'pollday.settings'
    Starting development server at http://127.0.0.1:5000/
    Quit the server with CONTROL-C.

  8. Почему команда создания проекта или приложения "ничего не пишет"?

    9. Комнада будет писать в случае скачивания пакетов установки, при необходимости, так же в случае ошибки, либо нужна специальная команда.
    В другом случае при запуске проекта может не выводиться ни одно сообщение

  9. Как проверить результат работы команд создания проекта или приложения?

    10. По структуре папок(появятся необходимые для приложения или проекта папки)
    или по коду в файлах.

  10. В каком файле пишем код на питоне для создания html-страниц?

    11. views.py
